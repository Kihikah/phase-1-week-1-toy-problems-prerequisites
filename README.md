# Week 1 Toy problems Prerequisites

## 1. Speed Detector

### Description
This program checks the speed of a car and determines whether the driver will receive a demerit point.

### How It Works
- If the speed is **less than or equal to 70 km/h**, the program outputs: `"Ok"`
- If the speed is above 70 km/h, every 5 km/h over the limit results in **1 demerit point**.
- If a driver accumulates **more than 12 points**, the program outputs: `"License suspended"`.

### Example Usage
#### Input:
```
Enter the speed of the car: 85
```
#### Output:
```
Points: 3
```

## 2. Net Salary Calculator

### Description
This program calculates an individual’s **Net Salary** by considering the following deductions:
- **PAYE (Tax)**
- **NHIF Deductions**
- **NSSF Deductions**
- **SHIF Deductions**
- **Housing Levy**

### How It Works
- The user enters their **basic salary** and **benefits**.
- The program computes **gross salary**.
- It applies the necessary deductions.
- The program then returns the **net salary**.

### Example Usage
#### Input:
```
Enter basic salary: 50,000
Enter benefits: 10,000
```
#### Output:
```
Gross Salary: 60,000
PAYE: 6,000
NHIF: 1,200
NSSF: 3,600
SHIF: 1,650
Housing Levy: 900
Total Deductions: 13,350
Net Salary: 46,650
```

## 3. Student Grade Generator

### Description
This program takes **student marks (0-100)** as input and assigns a grade according to the following scale:
- **A (80-100)**
- **B (60-79)**
- **C (50-59)**
- **D (40-49)**
- **E (0-39)**

### Example Usage
#### Input:
```
Enter student marks: 78
```
#### Output:
```
Grade: B
```

## How to Run These Programs
1. Open a web browser.
2. Open the Developer Console (Press `F12` or `Ctrl + Shift + J`).
3. Copy and paste the respective JavaScript code into the console.
4. Press `Enter` to run the program.
5. Follow the on-screen prompts to input data and get results.
